# 0.1.1 — 2026-06-25

First release carrying the complete gnomAD surface — seven tools (plus the opt-in `gnomad_dataframe_drop`), two resources, and the `gnomad_variant_triage` prompt over the keyless gnomAD GraphQL API, with ClinVar significance joined per variant. Per-tool detail is in the 0.1.0 entry; this release scopes the package and finalizes its publish metadata.

## Changed

- **npm scope** — the package name is now `@cyanheads/gnomad-genetics-mcp-server`. The npm/Cursor/VS Code install badges, the `bunx`/`npx` install args, the plugin manifests, and both `server.json` package identifiers use the scoped name. The reverse-domain `name` (`io.github.cyanheads/gnomad-genetics-mcp-server`) is unchanged.
- **Description** — one consistent one-line description across `package.json`, `server.json`, `manifest.json`, both plugin manifests, and the Docker OCI label.
- **`package.json`** — completed publish metadata: `mcpName`, `bugs`, `homepage`, `author`, `funding`, `packageManager`.
- **`manifest.json`** — added `repository`, `homepage`, and `license`.
- **`createApp()`** — dropped the redundant `websiteUrl`; the homepage derives from package metadata.
- **`@types/node`** (dev) — `^26.0.0 → ^26.0.1`.
